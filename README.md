# THINKQ bare-metal package

This package is prepared for **direct Linux deployment** on a university server or VM.

Start with:

- `REQUIREMENTS_AND_CLEANUP.md`
- `deploy/bare-metal/README.md`
- `BARE_METAL_DEPLOYMENT_NOTES.md`

## Included

- frontend source
- backend source
- bare-metal Nginx config
- systemd unit files
- environment file templates
- Aurora TLS notes

## Removed on purpose

- Docker
- Kubernetes
- tracked secrets
- local dependency directories
- local build artifacts
